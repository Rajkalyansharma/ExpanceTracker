# Expense Tracker App

A modern, intuitive Android application designed to help users track their daily expenses and manage their personal finances effectively. This app focuses on a clean user experience with offline-first capabilities.

## ✨ Features

* **Expense Management:** Easily add, edit, and delete your daily expenses.
* **Category Tracking:** Organize spending into categories (e.g., Food, Transport, Rent, Shopping).
* **Real-time Summaries:** View your total balance and spending trends at a glance.
* **Offline First:** All data is stored locally, ensuring the app works perfectly without an internet connection.
* **Clean UI:** Built with a modern, responsive interface for smooth navigation.

## 🛠 Tech Stack

* **Language:** [Kotlin](https://kotlinlang.org/)
* **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose) - Modern declarative UI.
* **Database:** [Room Persistence Library](https://developer.android.com/training/data-storage/room) - For robust local data storage.
* **Architecture:** MVVM (Model-View-ViewModel) - Ensuring separation of concerns and testability.
* **Asynchronous Programming:** Coroutines & Flow for reactive data handling.
* **Dependency Injection:** Hilt (or Koin) for clean architectural management.
